# Convert pa_matrix to data.frame

Convert pa_matrix to data.frame

## Usage

``` r
# S3 method for class 'pa_matrix'
as.data.frame(x, row.names = NULL, optional = FALSE, format = "long", ...)
```

## Arguments

- x:

  A `pa_matrix` object.

- row.names:

  Ignored.

- optional:

  Ignored.

- format:

  Character. "long" (default) or "wide".

- ...:

  Additional arguments (ignored).

## Value

A data.frame in long or wide format.
