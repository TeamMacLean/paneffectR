# Count singletons

Returns the number of singleton proteins in a clustering result.

## Usage

``` r
n_singletons(orthogroups)
```

## Arguments

- orthogroups:

  An `orthogroup_result` object.

## Value

Integer count of singletons.

## Examples

``` r
if (FALSE) { # \dontrun{
n <- n_singletons(clusters)
} # }
```
